## Getting Started

### Prerequisites
- Java Development Kit (JDK) 25 or higher
- IDE: IntelliJ IDEA or any Java-compatible IDE

### Building and Running

#### Compile the project:
```bash
javac src/test/Test.java
```

#### Run the main program:
```bash
java -cp src test.Test
```
