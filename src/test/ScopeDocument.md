Scope Document
Project Objective:
Develop a Java-based command line application that performs set operations on subsets (represented as boolean arrays) and multisets (represented as integer arrays) using a universal set (represented as string array) of n >= 10 elements.

Functional Requirements:
Part One:
Structure: Represent subsets as Boolean[]

Operations: Complement, Union, Intersection, Set Difference, and Symmetric Difference.

Output: Display raw bit array and the mapped element names

Part Two:
Structure: represent subsets as int[]

Operations: Multiset Union (Max), Intersection (Min), Difference (Subtraction with floor of 0), and Sum (Addition)

Output: Display raw integer array and the mapped element names

Restraints: Test cases must include at least two elements with a multiplicity > 1

Technical Specifications:
Language: Java (IntelliJ IDEA).

Universal Set: A static array containing at least 10 unique strings (e.g., fruits).

Testing: 2–3 representative test cases per module as well as at least 1 edge case (e.g., empty array)

Out of scope:
User input via scanner

Dynamic subset size

GUI

Success Criteria:
All 9 operations return accurate subset arrays

Subset arrays are correctly indexed to the universal set and printed

Thorough planning and documentation before/during implementation

Code is pushed to the Github Repository

A video < 10 minutes explaining the code and executing it

A PDF explaining implementation, verification, and answering the 6 analytical questions regarding natural language.
